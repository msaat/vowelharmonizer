# vowelharmonizer
A Java program that adds suffixes to Turkish nouns based on the rules of 2 and 4 way vowel harmony

the file twoVowel contains functions for adding suffixes which use 2 way vowel harmony
and the file fourVowel contains functions for adding suffixes which use 4 way vowel harmony

How it works:
- input a Turkish noun
- type out the name of suffix type you want to add from the given list
- the noun + corresponding suffix will be outputted
- if you wish to quit, enter 'exit' when prompted to quit the program
